# Lottery Statistical Analysis & Experimental AI Prediction System

## Overview

This project is designed to analyze 36 years of historical lottery data.

The purpose of this project is

- Statistical Analysis
- Pattern Mining
- Probability Analysis
- Machine Learning Experiment
- Deep Learning Experiment
- Visualization Dashboard

This project DOES NOT guarantee future lottery results.
